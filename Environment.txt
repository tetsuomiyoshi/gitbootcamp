Pythonの環境設定（Mac OS X版）

Pythonは３系の最新のものをインストールします。
(1)https://www.python.org/downloads/mac-osx/ のサイトから、
・Python3.7.0-2018-06-27の
・Download macOS 64-bit installer の
「macOS 64-bit installer」の部分を左クリックしてダウンロードする。
(2)Macの画面でDockのFinder（Dockの一番左にあるニコニコマーク）を左クリック
 -> Finder画面の左にある「ダウンロード」を左クリック
 -> ダウンロード画面の一番上にある「python-3.7.0-macosx10.9.pkg」をダブルクリック
 -> Pythonのインストーラーが出てくるので、その指示に従ってpythonをインストールする。
（注）基本的に、「続ける」、「同意する」を押して行けばいいです。途中で、マシンのパスワードを入れる箇所があります。

ターミナルを起動します。
(1)DockのLaunchpad（ロケットの絵）を左クリック
 -> その他を左クリック
 -> ターミナルを左クリック
 -> ターミナルの画面が現れ、自身のホームディレクトリに移動しています。
（注）ターミナルは今後使うことになるので、Dockにターミナルのアイコンが表示されていると思うので、それを右クリック->オプション->Dockに追加 を左クリックで、Dockに登録できます。今後はDockのターミナルを左クリックで起動できます。
（注）以下、自身のホームディレクトリに、cycloidのフォルダーがあるとして話を進めます。デスクトップにcycloidのフォルダーを置いている場合は、ターミナルから、cd Desktopと打ってretunキーを押して、Desktopにチェンジディレクトリしてから以下の操作を行ってください。
(2) >cd cycloid
（注）以下、>命令で、>（コマンドプロンプト:コマンドプロンプトは、ターミナル画面をどこでもいいので、左クリックしたときに現れる打鍵の位置を示すマーク（灰色の四角）のことです。）の後に命令を打って、returnキーを押す意味になります。
(3) >ls -la で必要ファイルなどがあることを確認してください。（animation1.py、animation2.py、animation3.pyなど）

Pythonが正しくインストールされているかを確認します。
(1) >python
Python 3.7.0〜と表示され、>>>が出ていればインストールは成功です。（Pythonが３系のものになっていることを確認してください。２系では駄目です。）
(2) >exit() で、Pythonの対話モードを抜けてください。（>>>の表示でなくなります。）

数値計算及びグラフィック用のPythonライブラリーをインストールします。
(1) >pip install numpy
インストールが成功すると、最後に、~successfulの文字が表示されます。
(2) >pip install matplotlib
インストールが成功すると、最後に、~successfulの文字が表示されます。
